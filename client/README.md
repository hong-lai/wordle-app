# Wordle App Client

A simple react app for wordle

## Features
- Play Wordle in a browser
- Default mode set to normal, please refer to the text-based app for more customizable configuration

## Quick Start
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
